Nuevo repo

…or create a new repository on the command line
echo "# hello-git" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/alberto-rosalesn/hello-git.git
git push -u origin main

Local a un repo nuevo

git branch -M main
